translate

  1. - filter in module gettext

Takes key and returns string

Sometimes it's not an option to use an attribute (e.g. when you want to annotate an attribute value). There's a translate filter available for this purpose.

<input type="text" placeholder="{{'Username'|translate}}" />

This filter does not support plural strings.

You may want to use custom annotations to avoid using the translate filter all the time.

Dependencies

  • gettextCatalog

Usage

In HTML Template Binding

{{ translate_expression | translate : context}}

In JavaScript

$filter('translate')(input, context)

Arguments

Param Type Details
input String

translation key
context String

context to evaluate key against

Returns

String

translated string or annotated key
