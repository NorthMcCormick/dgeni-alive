gettextPlurals

  1. - factory in module gettext

Provides correct plural form id for the given language

Example

gettextPlurals('ru', 10); // 1
gettextPlurals('en', 1);  // 0
gettextPlurals();         // 1

Usage

gettextPlurals(langCode, n);

Arguments

Param Type Details
langCode String

language code

(default: en)
n Number

number to calculate form for

(default: 0)

Returns

Number

plural form number
