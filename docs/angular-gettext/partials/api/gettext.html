gettext

Super simple Gettext for Angular.JS

A sample application can be found at https://github.com/rubenv/angular-gettext-example. This is an adaptation of the TodoMVC example. You can use this as a guideline while adding angular-gettext to your own application.

Installation

First include angular-gettext javascript file in your HTML:

You can download this file from the following places:

  • Bower
    e.g. 
    bower install angular angular-gettext

Then load the module in your application by adding it as a dependent module:

angular.module('app', ['gettext']);

With that you're ready to get started!

Provides correct plural form id for the given language
