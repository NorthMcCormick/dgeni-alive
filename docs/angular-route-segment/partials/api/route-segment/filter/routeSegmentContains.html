routeSegmentContains

  1. - filter in module route-segment

Check whether active segment contains the given segment name

Usage:

<li ng-class="{active: ('itemInfo' | routeSegmentContains)}">

Usage

In HTML Template Binding

{{ routeSegmentContains_expression | routeSegmentContains}}

In JavaScript

$filter('routeSegmentContains')(name)

Arguments

Param Type Details
name String

segment name

Returns

boolean

true if active segment contains given name
