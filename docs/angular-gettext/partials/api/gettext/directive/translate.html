translate

  1. - directive in module gettext

Annotates and translates text inside directive

Full interpolation support is available in translated strings, so the following will work as expected:

<div translate>Hello {{name}}!</div>

Dependencies

  • gettextCatalog
  • $parse
  • $animate
  • $compile
  • $window

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <translate
    [translate-plural="String"]
    [translate-n="Number"]
    [translate-context="String"]>
  ...
</translate>
  • as attribute: 
    <ANY
  translate
  [translate-plural="String"]
  [translate-n="Number"]
  [translate-context="String"]>
...
</ANY>

Arguments

Param Type Details
translate Boolean

node value is used as a translation key
translatePlural
(optional)
 String

plural form
translateN
(optional)
 Number

value to watch to substitute correct plural form
translateContext
(optional)
 String

context value, e.g. Verb, Noun

Events

  • gettextLanguageChanged

    Listens for language updates and changes translation accordingly

    Listen on: scope

Throws

  • Error

    MissingPlural for malformed plurals
