gettextFallbackLanguage

  1. - factory in module gettext

Strips regional code and returns language code only

Example

gettextFallbackLanguage('ru');     // "null"
gettextFallbackLanguage('en_GB');  // "en"
gettextFallbackLanguage();         // null

Usage

gettextFallbackLanguage(langCode);

Arguments

Param Type Details
langCode String

language code

Returns

StringNull

fallback language
