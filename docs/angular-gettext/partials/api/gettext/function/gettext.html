gettext

  1. - function in module gettext

Gettext constant function for annotating strings.

This function does nothing but it is used by angular-gettext-tools for building of a list of strings annotated in JavaScript code.

angular.module('myApp', ['gettext']).config(function(gettext) {
  /// MyApp document title
  gettext('my-app.title');
  ...
})

Usage

gettext(str);

Arguments

Param Type Details
str String

annotation key
