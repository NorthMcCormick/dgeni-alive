routeSegmentUrl

  1. - filter in module route-segment

Returns url for a given segment

Usage:

<a ng-href="{{ 'index.list' | routeSegmentUrl }}">
<a ng-href="{{ 'index.list.itemInfo' | routeSegmentUrl: {id: 123} }}">

Usage

In HTML Template Binding

{{ routeSegmentUrl_expression | routeSegmentUrl : params}}

In JavaScript

$filter('routeSegmentUrl')(name, params)

Arguments

Param Type Details
name String

fully qualified segment name
params Object

params to resolve segment

Returns

String

given url
