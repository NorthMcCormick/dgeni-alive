$routeSegmentProvider.options

  1. - type in module route-segment

Contains configuration options for $routeSegmentProvider

Properties

  • autoLoadTemplates

    Boolean

    When true, it will resolve templateUrl automatically via ($http)[https://docs.angularjs.org/api/ng/service/$http] service and put its contents into template.

  • strictMode

    Boolean

    When true, all attempts to call within method on non-existing segments will throw an error (you would usually want this behavior in production). When false, it will transparently create new empty segment (can be useful in isolated tests).
